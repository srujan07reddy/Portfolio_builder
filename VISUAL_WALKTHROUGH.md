## 🎯 USER JOURNEY MAP

```
START
  │
  ├──→ Home Page (/)
  │    • See features
  │    • Learn about platform
  │    • Click "View Demo" or "Get Started"
  │
  ├──→ Preview Dashboard (/dashboard/preview)
  │    • See portfolio in desktop view
  │    • Click "Mobile" button for mobile view
  │    • See real-time updates
  │
  ├──→ Edit Files
  │    • Edit profile.json (your content)
  │    • Edit theme.json (colors/template)
  │    • Changes show instantly in preview
  │
  ├──→ Publish
  │    • Click "Publish Site" button
  │    • Choose hosting provider
  │    • (Optional) Add custom domain
  │    • Click "Publish Now"
  │
  └──→ Live Portfolio
       Your site is now on the internet! 🎉
```


```
  └─ Your name, roles, subtitle
  └─ Bio and accomplishments
  └─ Projects and domains
  └─ Contact information

  └─ Choose template (Researcher/Minimalist/Corporate)
  └─ Choose color (Cyan/Violet/Emerald/Ruby)

  └─ Monitor with analytics
```

## 📊 WHAT HAPPENS WHEN YOU...

### Edit profile.json
```
profile.json modified
        ↓
Saved to disk
        ↓
Next.js detects change
        ↓
Hot reload triggered
        ↓
Preview updates instantly
```

### Click "Publish Site"
```
Modal opens
        ↓
Select hosting provider
        ↓
(Optional) Add domain
        ↓
Click "Publish Now"
        ↓
Status: Loading...
        ↓
Status: Success! ✅
        ↓
Your portfolio is LIVE
```

### Toggle Desktop/Mobile
```
Click [Desktop] or [Mobile]
        ↓
View changes instantly
        ↓
See responsive design
        ↓
Test on different sizes
```
