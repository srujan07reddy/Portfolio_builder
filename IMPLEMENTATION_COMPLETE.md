## 🚀 HOW TO USE

### Start Development Server:
```bash
cd "c:\Users\tagor\OneDrive\Desktop\Portfolio Builder\portfolio_builder"
npm run dev
```

### Access Points:
| Page | URL |
|------|-----|
| Landing | http://localhost:3000/ |
| Preview | http://localhost:3000/dashboard/preview |
| Login | http://localhost:3000/login |
| Signup | http://localhost:3000/signup |

### Customize:
1. Edit `src/content/profile.json` (your content)
2. Edit `src/content/theme.json` (colors/template)
3. Preview updates in real-time at `/dashboard/preview`

### Publish:
1. Click "Publish Site" button (bottom-right)
2. Choose hosting provider (Netlify recommended)
3. (Optional) Add custom domain
4. Click "Publish Now" → Live on internet!
